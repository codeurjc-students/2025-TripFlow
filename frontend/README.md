## 🚀 Getting Started

> To run the frontend of TripFlow, follow these steps:
> 
> 1. 📥 **Clone the Repository**:
>    ```bash
>    git clone https://github.com/codeurjc-students/2025-TripFlow.git
>    cd frontend
>    ```
> 
> 2. 📦 **Install Dependencies**:
>    ```bash
>    npm install
>    ```
> 
> 3. 🖥️ **Start the Development Server**:
>    ```bash
>    npm run dev
>    ```
